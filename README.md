# World Time App

An app made with Flutter. You can visualize different times for a bunch of locations. The source code can be found in the *lib/* directory.

When one clicks "Edit location" on the home screen it goes to the "Choose Location" screen. Then you can go back to the home screen either clicking the "<-" sign or choosing a location.
